Política de privacidade · Última atualização: maio de 2026

O que é coletado

Este site usa medição anônima de acessos para entender quais páginas são visitadas e como os visitantes chegam até ele. Nenhum dado pessoal é armazenado.

O que não é coletado

  • Nenhum cookie é criado
  • Seu endereço IP não é gravado — ele é descartado após ser convertido em um código anônimo
  • Nenhum perfil de usuário é construído ao longo do tempo
  • Nenhum dado é compartilhado com terceiros

Como funciona a anonimização

Cada visita gera um identificador temporário calculado a partir do seu IP, do seu navegador e de uma chave secreta que muda diariamente. O resultado é um código criptográfico que não permite reconstruir os dados originais. No dia seguinte, o mesmo navegador produz um código diferente.

Finalidade

Os dados agregados — como "quantas visitas esta semana" ou "de onde vieram os acessos" — são usados exclusivamente para avaliar o desempenho deste portfólio profissional.
